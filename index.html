Please follow the instructions to proceed the game

  1. Click start to begin the game
  2. Follow the pattern & click any colored circle among the
    6 different colored circles
  3. If the pattern we selected is correct then next level will start
  4. If we give the wrong pattern game is over and select check
    pattern to complete the level again
  5. Total 4 game levels.
